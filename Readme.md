#metacare

## Description

This is a test project that acts as a wrapper over swappi.dev, star wars repository. This project is written with typescript and has a database (postgres) SQL, and an ORM, (TypeORM)

## Installation

```bash
$ yarn
```

## Running the app

````bash
# development
$ yarn dev


## Test

```bash
# unit tests
$ yarn test
````
